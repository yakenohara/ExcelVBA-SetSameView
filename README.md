# 制限事項

* フォームの2回目以降の起動時は、1回目に起動した時にActiveになっていたBookが全面に表示されます。  
  ただし、フォームから設定する内容は起動の直前にActiveにしていたBookに対する設定です

* `To All Work Books`では、リボンの表示・非表示状態は反映されません
